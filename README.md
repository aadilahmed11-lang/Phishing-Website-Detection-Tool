PROJECT DESCRIPTION:
         This is a simple student project created using Python. The program checks whether a website link is safe or phishing.
PURPOSE:
        The main aim of this project is to help users identify fake websites that try to steal personal information like passwords and bank details.
TECHNOLOGIES USED:
       - Python
       - Pandas
       - Scikit-learn
       - Tkinter
HOW IT WORKS:
       1. The program uses sample website URLs.
       2. A machine learning model is trained using these URLs.
       3. Some basic rules are used to find suspicious words like login, verify, free.
       4. When the user enters a URL, the program checks it and shows the result.
HOW TO RUN:
       1. Install Python.
       2. Install required libraries.
       3. Run the Python file.
       4. Enter a website URL and click Check.
OUTPUT:
      The result will show whether the website is safe or phishing.
